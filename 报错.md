# 报错记录

## ✅ 已修复的错误

### 2025-08-11 视图文件缺失错误
**错误**: 模板文件不存在 (pending.html, approved.html, rejected.html)
**原因**: 重构控制器后未创建对应的视图文件
**修复**: 创建了3个专门的视图文件，支持不同状态的提现管理
**状态**: ✅ 已修复

### 2025-08-11 Outlay控制器访问级别错误
**错误**: Access level to app\admin\controller\Outlay::selectList() must be public
**原因**: 重构时错误地将selectList()方法设置为protected，但父类要求public
**修复**: 将selectList()方法访问级别改为public
**状态**: ✅ 已修复
