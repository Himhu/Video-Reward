## 🔧 性能优化措施

### 1. 权限查询优化
- 使用复合索引 `idx_module_status` 优化按模块和状态查询
- 角色权限关联使用外键约束，确保数据一致性
- 代理角色关联使用联合唯一索引，避免重复分配

### 2. 缓存策略
- AuthHelper支持权限缓存机制
- 前端权限验证脚本减少重复查询
- 权限树结构缓存，提升界面响应速度

### 3. 查询优化
```php
// 示例：优化的权限查询
public static function getAgentPermissions($agentId) {
    return self::alias('ar')
        ->join('roles r', 'ar.role_id = r.id')
        ->join('role_permissions rp', 'r.id = rp.role_id')
        ->join('permissions p', 'rp.permission_id = p.id')
        ->where('ar.agent_id', $agentId)
        ->where('r.status', 'active')
        ->where('p.status', 'active')
        ->field('p.*')
        ->select();
}
```
