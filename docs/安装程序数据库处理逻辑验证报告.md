## 🔍 1. 覆盖安装功能验证

#### **数据库存在性检查**
```php
function checkDatabase($database)
{
    $check = Db::query("SELECT * FROM information_schema.schemata WHERE schema_name='{$database}'");
    if (empty($check)) {
        return false;
    } else {
        return true;
    }
}
```

#### **数据库创建逻辑**
```php
function createDatabase($database)
{
    try {
        Db::execute("CREATE DATABASE IF NOT EXISTS `{$database}` DEFAULT CHARACTER SET utf8");
    } catch (\Exception $e) {
        return false;
    }
    return true;
}
```

### 🚨 **发现的严重问题**

#### **问题1：覆盖安装逻辑不完整** 🔴
**当前实现**：
- ✅ 检查数据库是否存在
- ❌ **缺少删除现有表的逻辑**
- ❌ **缺少清空数据的逻辑**
- ❌ **没有真正的"覆盖"处理**

**问题描述**：
当用户选择覆盖安装（cover=1）时，程序只是跳过数据库存在性检查，但并不会删除现有的表或数据。这意味着：
- 如果表已存在，CREATE TABLE语句会失败
- 现有数据不会被清空
- 可能导致表结构冲突
- 不是真正的"覆盖安装"

#### **问题2：表结构冲突处理缺失** 🔴
**当前实现**：
```sql
-- install.sql中使用的是标准CREATE TABLE语句
CREATE TABLE `ds_category` (
  `id` int(11) UNSIGNED NOT NULL AUTO_INCREMENT,
  -- ...
);
```

**问题描述**：
- 没有使用`CREATE TABLE IF NOT EXISTS`
- 没有`DROP TABLE IF EXISTS`语句
- 表已存在时会导致安装失败

#### **问题3：数据完整性保护不足** 🔴
**当前实现**：
- 没有备份现有数据的机制
- 没有回滚机制
- 覆盖安装风险极高

## 🔧 3. 问题修复方案

### 🚨 **紧急修复：覆盖安装功能**

#### **修复方案1：完善覆盖安装逻辑**
```php
/**
 * 处理覆盖安装
 */
function handleCoverInstall($database, $cover, $prefix = 'ds_')
{
    if ($cover == 1) {
        // 覆盖安装：删除现有表
        try {
            // 获取所有表名
            $tables = Db::query("SHOW TABLES LIKE '{$prefix}%'");
            
            // 删除现有表
            foreach ($tables as $table) {
                $tableName = array_values($table)[0];
                Db::execute("DROP TABLE IF EXISTS `{$tableName}`");
            }
            
            return true;
        } catch (\Exception $e) {
            return "覆盖安装失败：" . $e->getMessage();
        }
    }
    
    return true;
}
```

#### **修复方案2：优化install.sql结构**

#### **修复方案3：增强错误处理**
```php
function install($admin_username, $admin_password, $config, $adminUrl)
{
    try {
        // 处理覆盖安装
        $coverResult = handleCoverInstall($config['database'], $config['cover'], $config['prefix']);
        if ($coverResult !== true) {
            return $coverResult;
        }
        
        // 执行SQL安装
        $sqlFile = __DIR__ . '/config/install/sql/install.sql';
        if (!file_exists($sqlFile)) {
            return 'SQL文件不存在';
        }
        
        $sql = file_get_contents($sqlFile);
        $sqlList = parseSql($sql, $config['prefix'], 'ds_');
        
        foreach ($sqlList as $v) {
            $v = trim($v);
            if (empty($v)) continue;
            
            try {
                Db::execute($v);
            } catch (\Exception $e) {
                return "SQL执行失败：" . $e->getMessage() . " SQL: " . $v;
            }
        }
        
        // 创建管理员账户
        $result = createAdminUser($admin_username, $admin_password);
        if ($result !== true) {
            return $result;
        }
        
        // 生成配置文件
        $result = createConfigFile($config);
        if ($result !== true) {
            return $result;
        }
        
        return true;
        
    } catch (\Exception $e) {
        return "安装过程发生错误：" . $e->getMessage();
    }
}

/**
 * 验证表前缀格式
 */
function validatePrefix($prefix)
{
    // 检查前缀格式
    if (!preg_match('/^[a-zA-Z][a-zA-Z0-9_]*_$/', $prefix)) {
        return '表前缀格式不正确，必须以字母开头，以下划线结尾';
    }
    
    // 检查长度
    if (strlen($prefix) > 20) {
        return '表前缀长度不能超过20个字符';
    }
    
    return true;
}
```
