### 2. **getInstallerUrl() 方法简化**

#### 修改前 (22行复杂逻辑):
```php
/**
 * 获取安装程序URL
 * 
 * 优先使用重构版安装程序，向后兼容原版安装程序
 * 
 * @return string 安装程序URL
 */
public function getInstallerUrl(): string
{
    $publicPath = $this->appPath . 'public' . DIRECTORY_SEPARATOR;
    
    // 检查重构版安装程序
    $newInstaller = 'install_new.php';
    if (file_exists($publicPath . $newInstaller)) {
        return $this->buildUrl($newInstaller);
    }
    
    // 检查原版安装程序
    $oldInstaller = 'install.php';
    if (file_exists($publicPath . $oldInstaller)) {
        return $this->buildUrl($oldInstaller);
    }
    
    // 默认返回重构版（即使文件不存在，也提供明确的错误信息）
    return $this->buildUrl($newInstaller);
}
```

#### 修改后 (4行简洁逻辑):
```php
/**
 * 获取安装程序URL
 * 
 * 使用重构版安装程序
 * 
 * @return string 安装程序URL
 */
public function getInstallerUrl(): string
{
    $installer = 'install_new.php';
    return $this->buildUrl($installer);
}
```
