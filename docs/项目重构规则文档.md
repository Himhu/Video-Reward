## 7. 代理管理系统设计

### 7.1 用户角色重新定义
- **超级管理员** (admin.id = 1)
  - 系统最高权限
  - 可管理所有代理账户
  - 查看全局数据统计
  - 系统配置管理权限

- **代理用户** (admin.id ≠ 1)  
  - 业务操作权限
  - 只能查看自己的数据
  - 设备指纹身份识别
  - 受权限控制约束

### 7.2 权限控制设计
```php
// 权限控制示例
class PermissionService
{
    /**
     * 检查用户是否为超级管理员
     */
    public function isSuperAdmin($userId): bool
    {
        return $userId === 1;
    }
    
    /**
     * 检查用户数据访问权限
     */
    public function canAccessData($userId, $dataOwnerId): bool
    {
        return $this->isSuperAdmin($userId) || $userId === $dataOwnerId;
    }
}
```

// 缓存使用示例
class CacheService
{
    /**
     * 获取缓存数据
     */
    public function get($key, $default = null)
    {
        return cache($key) ?: $default;
    }

    /**
     * 设置缓存数据
     */
    public function set($key, $value, $ttl = 3600): bool
    {
        return cache($key, $value, $ttl);
    }

    /**
     * 删除缓存
     */
    public function delete($key): bool
    {
        return cache($key, null);
    }
}
