## 🔒 代理管理系统设计

### 用户角色重新定义
- **超级管理员** (admin.id = 1)
  - 系统最高权限
  - 可管理所有代理账户
  - 查看全局数据统计
  - 系统配置管理权限

- **代理用户** (admin.id ≠ 1)
  - 业务操作权限
  - 只能查看自己的数据
  - 设备指纹身份识别
  - 受权限控制约束

### 权限控制机制
```php
// 权限控制示例
class PermissionService
{
    public function isSuperAdmin($userId): bool
    {
        return $userId === 1;
    }
    
    public function canAccessData($userId, $dataOwnerId): bool
    {
        return $this->isSuperAdmin($userId) || $userId === $dataOwnerId;
    }
}
```
